Behavior Buddy Board helps teachers run behavior coaching on a smartboard:

- Load and save your student list once
- Keep every student visible all day
- Move students between Green / Yellow / Red zones in one tap
- Mark "Star Day" moments for positive reinforcement (and clear stars for a new day)
- Swap each student's Classroom Zoo buddy avatar directly on the board
- Uses the same Classroom Zoo character cast for brand consistency
