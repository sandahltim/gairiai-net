Glyph Cathedral turns a plain phrase into a vertical artifact. Each bay in the structure swaps your letters through curated Unicode families, so the result feels half sacred manuscript, half software hallucination. The point is not just that it looks good. The point is that language can bend into architecture without stopping being readable.

The best part is the switch from Beauty mode to Threat mode. In Beauty mode, the substitutions stay ceremonial and traceable. In Threat mode, the same phrase leans toward phishing-grade doubles and near-invisible script shifts, which turns the whole monument into a lesson about how easy it is to hide danger inside familiar-looking text.

You can type your own phrase, shuffle the glyph families, run a cinematic reveal, and export a poster-sized PNG. It is a typography instrument, a miniature security demo, and a weirdly emotional piece of generative architecture all at once.
