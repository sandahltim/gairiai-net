Title and category: Pixel Art Studio (daily)

One-paragraph description: Pixel Art Studio is a fully-featured, self-contained pixel art editor that lets users draw on an infinite canvas, experiment with a customizable 16-color palette, and export their creations as PNG images or ASCII art. Perfect for quick sketches, game sprites, or retro-style artwork directly in the browser.

Target audience: Aspiring artists, game developers, retro gaming fans, and anyone who enjoys creative, bite-sized digital art sessions.

Key interactions/features:
- Infinite scrollable and zoomable canvas using HTML Canvas.
- Toolbar with pencil, bucket fill, eraser, selection/move, and line tools.
- Symmetry drawing mode (horizontal/vertical/diagonal) for mirrored creations.
- Color picker and palette editor with 16 swatches.
- Export buttons for PNG download and ASCII art preview/copy.

Why this will be interesting: Pixel art evokes nostalgia for 8-bit era games while offering modern fluidity; the instant export makes sharing effortless on social media; it's endlessly replayable with no save needed, and the symmetry mode adds a "wow" factor for complex patterns quickly. Educational on canvas API basics, highly viral potential with user screenshots.
